# 🚀 Squad Battle Management System

A fully interactive web-based squad battle simulator built with **HTML, CSS, and Vanilla JavaScript**.

This project includes:

- 🧬 Squad attribute management  
- 👾 Real-time alien battlefield combat  
- 🗺️ Arena map effects system  
- ⚔️ Squad vs Squad arena comparison  
- 🎒 Item wall assignment system  
- 💰 Betting & reward system  
- 📊 Dynamic HP and points control  
- 🌌 Animated sci-fi UI design  

---

## 🎮 Project Overview

This project is an interactive squad battle simulation system designed for:

- Strategy games  
- Classroom battle simulations  
- Team competitions  
- Game prototyping  
- Frontend development practice  

The system runs entirely in a single `index.html` file and does not require any external libraries or frameworks.

---

## 🧩 Main Features

### 1️⃣ Squad Management System

Users can create and manage multiple squads with customizable attributes:

- ❤️ HP  
- ⚔️ Attack  
- 🛡️ Defense  
- ⚡ Speed  
- 🎯 Accuracy  
- ✨ Special Ability  
- 💎 Total Power (auto-calculated)  

Each squad card supports:

- Editable numeric inputs  
- Animated attribute bars  
- Real-time total power calculation  

---

### 2️⃣ Alien Battlefield System

A visual battlefield where squads attack alien enemies.

Features include:

- 10 animated floating alien units  
- Editable HP system  
- Reward points tracking  
- Attack count per squad  
- Dead state visualization (grayscale + glow)  
- Target selection highlighting  
- Manual HP & points adjustment tools  
- Reset functionality  

---

### 3️⃣ Arena Map System

Different arena maps apply battle modifiers.

Each arena:

- Has a unique icon  
- Displays special effects  
- Applies boost or nerf modifiers  
- Shows effect descriptions  

Effect Types:

- 🔴 Zero Effect  
- 🟡 Nerf  
- 🟢 Boost  

---

### 4️⃣ Squad vs Squad Arena Comparison

Compare two squads attribute-by-attribute.

System Logic:

- Compare each attribute  
- Assign points per winning stat  
- Highlight winners  
- Display final victory result  
- Handle tie cases  
- Reset functionality  

Includes:

- Total power comparison  
- Detailed attribute breakdown  
- Animated victory display  

---

### 5️⃣ Item Wall System

A draft-style item assignment system.

Features:

- Item wall grid  
- Click-to-select system  
- Assign item to squad  
- Track taken items  
- Undo last assignment  
- Disable already taken items  

---

### 6️⃣ Betting System 💰

Adds a competitive spectator mode.

Features:

- Select two fighting squads  
- Other squads place bets  
- Betting odds display  
- Reward calculation  
- Winner announcement  
- Rewards distribution  
- Clear bets functionality

## ⚙️ How to Run

### Option 1 — Local

1. Download or clone the repository  
2. Open `index.html` in your browser  

No installation required.

### Option 2 — GitHub Pages

1. Push the project to GitHub  
2. Enable GitHub Pages  
3. Select the `main` branch  
4. Access your live site
